# Roadmap

## v1.0 Community Edition

- Stable local and Docker setup.
- Portfolio transaction entry and CSV import.
- Portfolio analytics, allocation drift, diagnostics, alerts, and AI summaries.
- Public documentation for architecture, API, and database schema.

## Future Improvements

- JWT-based authentication and session expiry.
- Alembic migrations for database evolution.
- Background workers for scheduled price and news refresh.
- Playwright smoke tests for frontend flows.
- More granular provider abstraction tests.
